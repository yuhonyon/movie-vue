## 安裝

``` bash
# 安裝一來
npm install

# 开发模式
npm run dev

# 打包
npm run build

# analyzer report
npm run build --report

# unit 测试
npm run unit

# e2e 测试
npm run e2e

# 所有测试
npm test
```
